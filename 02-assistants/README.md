For an in-depth walkthrough, check out the [recording](https://vimeo.com/990334325/56b552bc7a).

## Setup

1. Set your OpenAI key.

```bash
export OPENAI_API_KEY="sk_XXX..."
```

2. Run.

```bash
python 7_orchestration.py
```
